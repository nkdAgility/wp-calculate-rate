Adds a shortcode for displaying Rate and an options page for setting it

== Description ==

Its hard to keep your rate in sync accross you site so this plugin adds two 
shortcodes and one options box. Options page lets you add your "Rate" in 
whatever currancy your like,then the [nkd-rate] adn [nkd-calculate] shortcodes
allow you to displa it how you like
